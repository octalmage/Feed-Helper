This plugin helps fix a common rss validation error. 

== Description ==

This plugin finds newlines at the end of plugin files that commonly causes the "wordpress error on line 2 at column 6: XML declaration allowed only at the start of the document" error.

To use the plugin, click "Feed Helper" under the tools section, then press start. 

Feed Helper will then scan your plugins.
